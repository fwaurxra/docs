## What is RestifyX.js?

RestifyX.js is a modern, feature-rich framework for building RESTful APIs in Node.js. It combines the flexibility of Express with powerful abstractions that make API development faster and more enjoyable.

## Key Features

RestifyX.js comes packed with features to accelerate your API development:

- **Fluent API Design** - Intuitive, chainable methods for defining endpoints
- **Decorator Support** - Use TypeScript decorators for clean, declarative code
- **Built-in Validation** - Validate requests with minimal setup
- **Automatic Swagger Docs** - API documentation generated automatically
- **Hot Reloading** - Instant updates during development
- **Performance Monitoring** - Built-in tools to track API performance
- **Security Features** - Robust security with sensible defaults
- **WebSocket Support** - Real-time communication made easy
- **Middleware System** - Flexible middleware architecture
- **Error Handling** - Comprehensive error management

## Why RestifyX.js?

RestifyX.js was built to solve common pain points in API development:

- **Reduce Boilerplate** - Focus on business logic, not repetitive code
- **Type Safety** - First-class TypeScript support
- **Developer Experience** - Intuitive APIs and helpful error messages
- **Performance** - Optimized for high-throughput applications
- **Flexibility** - Adapt to your project's needs without constraints
- **Production Ready** - Built with real-world applications in mind

## Quick Example

\`\`\`typescript
import { create, CreateEndpoint, SendData } from 'restifyx.js';

// Create the application
const app = create({
  port: 3000,
  debug: true
});

// Define an endpoint using the fluent API
const helloEndpoint = CreateEndpoint()
  .setName('Hello World')
  .setPath('/hello/:name')
  .setMethod('GET')
  .setDescription('Says hello to the provided name')
  .addParams({ name: 'string' })
  .setHandler(async (header, body, query, cookie, params) => {
    SendData.json({
      message: `Hello, ${params.name}!`,
      timestamp: new Date()
    });
  });

// Register the endpoint manually
app.registerEndpoint(helloEndpoint.build());

// Start the server
async function start() {
  await app.setup();
  await app.start();
  console.log('Server running on http://localhost:3000');
}

start().catch(console.error);
